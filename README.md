# Mercury API (JS + TS, CommonJS)

This monorepo contains two backend APIs developed side-by-side in JavaScript and TypeScript using TDD, Fastify, and Vitest.

## Apps
- `apps/mercury-js`: CommonJS JavaScript implementation
- `apps/mercury-ts`: CommonJS TypeScript implementation

## Commands
Run tests:
```bash
cd apps/mercury-js && npm test
cd apps/mercury-ts && npm test
